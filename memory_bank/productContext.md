# Product Context

## Название продукта
**LinkLayer** — умный блокнот с системой графов для управления знаниями

## Проблема, которую решает продукт

### Основные боли пользователей
1. **Сложность организации информации**: Большие объемы информации сложно структурировать и найти нужное
2. **Отсутствие визуализации связей**: Сложно видеть, как идеи связаны между собой
3. **Сложность инструментов**: Obsidian мощный, но сложный для новичков
4. **Отсутствие совместной работы**: Нет простого способа работать над знаниями в команде
5. **Привязка к одному устройству**: Данные не синхронизируются между устройствами

## Целевая аудитория

### Первичная аудитория
- **Все желающие** (студенты, исследователи, писатели, профессионалы)
- Люди, которые хотят организовать свои знания
- Люди, которые ценят визуализацию информации
- Люди, которые хотят простой и понятный инструмент

### Вторичная аудитория
- Небольшие компании (2-50 человек)
- Инди-разработчики и фрилансеры
- Исследовательские группы
- Образовательные учреждения

## Ценностное предложение

### Для индивидуальных пользователей
- ✅ **Простота**: Интуитивный UI, понятнее чем Obsidian
- ✅ **Визуализация**: Два режима графов для разных стилей работы
- ✅ **Доступность**: Веб-приложение, доступно с любого устройства
- ✅ **Гибкость**: Ручное соединение заметок, группировка, поиск по тегам
- ✅ **Экспорт**: Возможность импорта/экспорта .md файлов

### Для команд
- ✅ **Совместная работа**: Общие хранилища для команд
- ✅ **Синхронизация**: Изменения видны всем в реальном времени
- ✅ **Коммуникация**: Встроенный чат с упоминанием заметок
- ✅ **Масштабируемость**: Может помочь небольшим компаниям и инди-разработчикам

## Конкурентный анализ

### Основные конкуренты
1. **Obsidian**
   - Сильные стороны: Мощный, offline-first, большое сообщество
   - Слабые стороны: Сложный UI, нет встроенной совместной работы, платный для синхронизации
   - Наше преимущество: Проще UI, встроенная совместная работа, бесплатно

2. **Notion**
   - Сильные стороны: Красивый UI, встроенная совместная работа, много интеграций
   - Слабые стороны: Сложный для новичков, дорогой, медленный
   - Наше преимущество: Проще, быстрее, ориентирован на графы

3. **Roam Research**
   - Сильные стороны: Граф-ориентированный, мощный поиск
   - Слабые стороны: Дорогой, сложный UI, медленный
   - Наше преимущество: Дешевле, проще, быстрее

### Наше уникальное предложение
- **Два режима графов**: Классический (как Obsidian) и физический (как доска детектива)
- **Простота**: Интуитивный UI, не перегруженный функциями
- **Встроенная совместная работа**: Не нужно платить за синхронизацию
- **Бесплатно**: Для MVP и базовых функций

## Бизнес-модель

### MVP (Фаза 1)
- **Бесплатно** для всех пользователей
- Сбор минимума данных (username, способ регистрации)
- Цель: Получить пользователей и обратную связь

### Будущие версии (Фаза 2+)
- **Freemium модель**:
  - Бесплатный план: Базовые функции, ограничение на размер хранилища
  - Платные планы: Расширенные функции, больше места, приоритетная поддержка
  - Командные планы: Для компаний, расширенные права доступа

## Метрики успеха

### Для MVP
- Количество зарегистрированных пользователей
- Количество созданных хранилищ
- Количество заметок в среднем на пользователя
- Активность пользователей (DAU, MAU)
- Удержание пользователей (retention rate)

### Для совместной работы
- Количество совместных хранилищ
- Среднее количество участников в хранилище
- Активность в чате
- Количество комментариев

## Видение на 5 лет

LinkLayer станет **стандартным инструментом для управления знаниями** для:
- Студентов и исследователей
- Профессионалов, которые работают с информацией
- Команд, которые хотят совместно работать над знаниями
- Компаний, которые хотят документировать свои процессы

Мы хотим быть **проще и доступнее** чем Obsidian, но **мощнее** чем простые заметки.

## Ограничения и предположения

### Предположения
- Пользователи хотят простой инструмент, а не мощный
- Пользователи ценят визуализацию информации
- Пользователи готовы хранить данные на сервере
- Команды хотят простого способа совместной работы

### Ограничения
- Только веб-версия для MVP (мобильная позже)
- Без шифрования данных на сервере
- Без истории версий для MVP
- Без встроенных изображений/видео

## Дорожная карта

### Фаза 1: MVP (текущая)
- Личные хранилища
- Система заметок и тегов
- Два режима графов
- Базовая совместная работа (добавление соавторов, чат)
- Импорт/экспорт .md

### Фаза 2: Расширенные функции
- История версий
- Встроенные изображения
- Система комментариев
- Расширенные типы связей

### Фаза 3: Полная совместная работа
- Система ролей (owner, editor, viewer)
- Разрешение конфликтов при одновременном редактировании
- Расширенный чат с потоками
- Уведомления о активности

### Фаза 4: Интеграции
- Экспорт в JSON, CSV
- Синхронизация с облачными хранилищами
- API для интеграций

### Фаза 5: Мобильность
- Мобильная версия
- Offline-first синхронизация

### Фаза 6: Монетизация
- Платные планы
- Командные планы для компаний
